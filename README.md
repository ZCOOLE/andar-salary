# 员工绩效薪资管理系统

这是一个基于 React 18 + TypeScript + Tailwind CSS 开发的员工绩效薪资管理系统，旨在帮助企业管理员工绩效评估和薪资发放流程。

## 项目功能

### 核心模块

1. **员工信息管理**
   - 员工信息的添加、编辑、删除和查询
   - 支持员工基本信息、联系方式、银行卡信息等管理
   - 支持按姓名、工号、部门等搜索

2. **薪资数据管理**
   - 薪资历史查询
   - 薪资明细查看
   - 薪资计算（基本工资、绩效奖金、扣除项等）

3. **绩效评估**
   - 员工绩效自评
   - 领导批量评估
   - 绩效详情查看
   - 绩效状态跟踪

4. **财务导出**
   - 公司薪资汇总导出
   - 银行转账文件导出
   - 支持 CSV 格式

5. **权限控制**
   - 角色分级访问（员工、领导、财务）
   - 基于角色的功能权限

6. **用户认证**
   - 登录验证（当前为模拟登录，支持一键登录）
   - 角色识别
   - 会话管理

### 技术栈

- **前端框架**：React 18 + TypeScript
- **样式框架**：Tailwind CSS v4
- **路由管理**：React Router v7
- **数据存储**：LocalStorage（本地存储）
- **导出功能**：CSV 导出
- **图标库**：Lucide React

## 运行项目

### 安装依赖

```bash
npm i
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

## 使用流程

### 员工流程
1. 登录系统
2. 查看个人首页，了解待办事项
3. 完成绩效自评
4. 查看绩效评估结果
5. 查询历史薪资记录
6. 查看个人信息

### 领导流程
1. 登录系统
2. 查看团队绩效待办
3. 批量或逐个评估员工绩效
4. 查看部门绩效汇总
5. 查询团队薪资情况
6. 管理员工信息（如需）

### 财务流程
1. 登录系统
2. 查看薪资数据概览
3. 导出薪资汇总和银行转账文件
4. 管理员工信息（如需）

## 数据管理

当前项目使用 LocalStorage 进行数据存储，数据仅保存在本地浏览器中，适合演示和小规模使用。

### 数据结构

- **员工数据**：包含基本信息、联系方式、银行卡信息等
- **绩效数据**：包含自评分数、领导评分、评语等
- **薪资数据**：包含基本工资、绩效奖金、扣除项等
