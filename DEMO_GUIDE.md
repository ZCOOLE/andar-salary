# 员工绩效薪资管理系统 - 演示指南

## 🎯 系统简介

这是一个功能完整的员工绩效薪资管理系统演示原型，支持三种角色：
- **员工**：绩效自评、查看薪资
- **领导**：绩效评估、团队管理
- **财务**：薪资管理、数据导出

## 📱 功能演示路径

### 1️⃣ 员工角色体验（张三账号）

1. 登录后查看首页
   - 查看本月绩效状态
   - 如果显示"待自评"，点击"立即填写"

2. 绩效自评
   - 调整评分（0-100分）
   - 填写工作成果描述
   - 可以"保存草稿"或"提交自评"

3. 查看薪资
   - 点击底部"薪资"Tab
   - 查看历史薪资记录
   - 点击任意记录查看详情

4. 个人中心
   - 点击底部"我的"Tab
   - 查看个人信息
   - 可以退出登录

### 2️⃣ 领导角色体验（李经理账号）

1. 登录后查看首页
   - 查看待评估员工数量
   - 点击"一键审批"或"逐个审批"

2. 批量评估
   - 勾选需要评估的员工
   - 默认跟随员工自评分
   - 可以单独修改评分和评语
   - 点击"批量通过"完成评估

3. 逐个评估
   - 查看员工自评详情
   - 调整领导评分
   - 填写评语
   - 查看绩效金额预览
   - 提交评估

4. 团队管理
   - 查看绩效页面的所有员工
   - 查看薪资页面的部门数据
   - 查看统计信息

### 3️⃣ 财务角色体验（王财务账号）

1. 登录后查看首页
   - 查看薪资确认状态
   - 查看待确认人数

2. 薪资管理
   - 查看所有员工薪资
   - 查看总发放金额
   - 查看平均绩效系数
   - 点击员工查看详细薪资构成

3. 数据导出（演示）
   - 点击"导出报表"查看提示

## 🎨 界面特色

- **移动端优先设计**：适配手机屏幕，支持触摸操作
- **底部Tab导航**：快速切换主要功能
- **渐变色卡片**：美观的视觉效果
- **实时数据计算**：绩效金额自动计算
- **本地数据持久化**：使用localStorage保存数据

## 💾 数据说明

- 所有数据保存在浏览器本地存储（localStorage）
- 切换账号可以体验不同角色的功能
- 数据修改会实时保存
- 清除浏览器缓存会重置所有数据

## 📝 技术栈

- **前端框架**：React 18 + TypeScript
- **路由**：React Router v7
- **样式**：Tailwind CSS v4
- **图标**：Lucide React
- **状态管理**：useState + localStorage
- **构建工具**：Vite

## 🔄 数据流程

```
员工自评 → 领导评估 → 绩效确认 → 薪资计算 → 工资发放
   ↓          ↓          ↓          ↓          ↓
 待自评    待评估    已完成    财务确认    已发放
```
